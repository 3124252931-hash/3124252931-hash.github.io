NO₂ Monitoring Stations in Glasgow

This interactive map visualises annual mean nitrogen dioxide (NO₂) concentrations measured at air quality monitoring stations across Glasgow.

Each purple circle represents a monitoring station. The size of the circle increases with higher NO₂ concentration (µg/m³).

Use the year selector to explore changes in NO₂ levels over time.

Data source: UK Air Quality Monitoring Network
